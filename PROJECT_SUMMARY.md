# Project Summary

## Overview
Bookly is a Flutter application designed to browse and search for books using the Google Books API. It utilizes Clean Architecture and Cubit for state management.

## Architecture
- **Pattern**: Clean Architecture (Presentation, Data).
- **State Management**: Bloc/Cubit.
- **Navigation**: GoRouter.
- **Networking**: Dio with Dartz for error handling.

## Features
- **Home**: Featured and Newest books feed.
- **Search**: Book search functionality.
- **Book Details**: Detail view for individual books.

## Progress
- **Phase**: Initial documentation and feature exploration.
- **Current Status**: README.md created. Project structure and core features identified.
